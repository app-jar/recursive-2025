# Объяснение решения задачи

Дано правило на языке Пролог для проверки, что один список получается из другого путём попарного обмена соседних элементов.

pair_reverse([X,Y|T1], [Y,X|T2]) :-
pair_reverse(T1, T2).

Описание правила:
- Берутся первые два элемента списка: `[X, Y]`
- Меняются местами: `[Y, X]`
- Рекурсивно вызывается `pair_reverse` для оставшейся части списка `T1`
- Результат присваивается `T2`

Таким образом, функция проверяет, что второй список получен из первого путём попарного обмена соседних элементов.
