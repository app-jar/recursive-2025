# Алгоритм деления в столбик

## Описание

Реализация алгоритма деления в столбик с евклидовой семантикой. Функция возвращает частное и остаток.

## Формат результата

Функция `long-division` возвращает список из двух элементов:
- `quotient` - частное
- `remainder` - остаток (всегда неотрицательный, 0 ≤ remainder < |divisor|)

## Евклидово деление

Алгоритм реализует евклидово деление, где:
- Остаток всегда неотрицательный: 0 ≤ remainder < |divisor|
- Выполняется основное тождество: dividend = divisor × quotient + remainder

Примеры:
- -17 ÷ 5 = -4 (остаток 3), так как -17 = 5 × (-4) + 3
- 17 ÷ -5 = -4 (остаток -3), так как 17 = -5 × (-4) + (-3)
