В коде p1 и p2 это списки коэффициентов многочленов, где элемент с индексом 0 отвечает за коэффициент при x^0, с индексом 1 — за х^1 и так далее. 
Результат функции: новый список коэффициентов, соответствующий произведению двух многочленов по степеням и коэффициентам.
Пример:
(poly-multiply '(1 2 3) '(4 5))  ;; (1 + 2x + 3x^2) * (4 + 5x)
;; => (4 13 22 15) => 4 + 13x + 22x^2 + 15x^3
