# Лабораторная работа 1, Задача 21 (Common Lisp)

## Условие
По заданному списку посчитать количество каждого из встречаемых в нём элементов.

## Решение
Программа рекурсивно подсчитывает частоту встречаемости каждого элемента в списке.

## Алгоритм
1. Используется рекурсивная функция с аккумулятором
2. Для каждого элемента проверяется, встречался ли он ранее
3. Если элемент уже встречался - увеличивается счетчик
4. Если элемент новый - добавляется в результат с счетчиком 1
5. Используется assoc для поиска элементов в списке пар
