# Лабораторная работа 4, Задача 19 (Common Lisp)

## Условие
Получить все возможные разложения заданного числа в виде суммы. Перестановка слагаемых нового способа не даёт.

## Решение
Программа генерирует все возможные разбиения числа на слагаемые по возрастанию.

## Алгоритм
1. Используется рекурсивный подход с ограничением максимального слагаемого
2. Для избежания дубликатов слагаемые располагаются в невозрастающем порядке
3. Каждое следующее слагаемое не превышает предыдущее
