# Лабораторная работа 1, Задача 13

## Условие
Является ли один список перестановкой элементов другого списка, учитывая сколько раз встретился каждый из элементов.

## Решение
Программа проверяет что:
1. Списки имеют одинаковую длину
2. Содержат одинаковые множества элементов  
3. Частоты встречаемости каждого элемента совпадают

## Алгоритм
1. Сравниваем длины списков
2. Сортируем списки и сравниваем множества элементов
3. Подсчитываем частоты каждого элемента
4. Сравниваем распределения частот
