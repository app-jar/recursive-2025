# recursive-2025
Лабораторные работы по РЛП+ФП 2025

## Порядок сдачи
1. Создаем ветку `<Фамилия><Имя>_task-<НомерЛабы>_op-<НомерЗадачи>`. (e.g. `DubenkoIlia_task-1_op-12`)
2. Создаем папку `<Фамилия><Имя>` в папке группы, если нет такой. (e.g. `group-9.1/DubenkoIlia`)
3. В своей папке добавляем папку `task-<НомерЛабы>_op-<НомерЗадачи>` и в нее кладем файл с решением (e.g. `group-9.1/DubenkoIlia/task-1_op-12/task.hs`). В папку задачи можно положить дополнительные артефакты (readme и тп)
4. Пушим ветку с решением в Гитхаб. Создаем Pull Request, в ревьюеры `app-jar` и 2х одногруппников.
5. После того, как 2 одногруппника зааппрувят PR, его посмотрит Преп.
6. Преп оставит комменты в PR. После того, как студет их все разрешит - преп заапрувит изменения и вольет ветку. На этом задачу можно считать принятой

## Оформление работы
Код должен быть чистым и понятным, с **необходимыми** комментами (без фанатизма!!!)

В начале файла - коммент с номером задачи и условием

В коммите указываем нормальный понятный месседж. Если это первый коммит по лабе, то укажите номер и условие задачи

```
Номер 2, задача 21
Эмуляция умножения в столбик
```

## Что еще
Проверяйте почаще гитхаб. Дейлайте друг другу код ревью. 

Если что, одногруппники тоже могут открывать треды с комментами. Но преп приступит к проверке, только когда все треды будут разрешены и одногруппники заапувяд PR.

**В случае некачественного ревью к одногруппникам будут применены санкции) Так что кнопочку аппрув жмем только когда нет вопросов**
