# recursive-2025
Лабораторные работы по РЛП+ФП 2025

## Порядок сдачи
1. Создаем ветку <Фамилия><Имя>_task-<НомерЛабы>_op-<НомерЗадачи>. (e.g. DubenkoIlia_task-1_op-12)
2. Если нет, то создаем папку <Фамилия><Имя> в корне репозитория. (e.g. DubenkoIlia)
3. В своей папке добавляем папку task-<НомерЛабы>_op-<НомерЗадачи> и в нее кладем файл с решением (e.g. DubenkoIlia/task-1_op-12/task.hs). В папку задачи можно положить дополнительные артефакты (readme и тп)
